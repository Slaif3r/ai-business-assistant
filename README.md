# Python / AI Business Assistant

## Overview
This project is an AI-powered business intelligence application for small businesses.

It provides:
- Sales forecasting
- Customer sentiment analysis
- Data-driven insights for decision-making

## Target Users
- Small business owners
- Shop managers
- Restaurant owners

## Core Value
Help businesses move from intuition-based decisions to data-driven decisions.	


Intial Stack:

| Purpose|Tool|
|---|---|
| Language| Python 3.12|
| IDE| VS Code|
|UI| Streamlit|
|ML| Scikit-learn|
|DL| PyTorch|
|NLP| scikit-learn + PyTorch|
|Charts| Plotly|
|Version control| Git + GitHub|
|Env| venv|
|Docs| Markdown|
|Deployment | Streamlit Cloud|
